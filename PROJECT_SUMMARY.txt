1. PROJECT OVERVIEW
================================================================================

PROJECT NAME: Lumina
TAGLINE: Your Sacred Sanctuary
TYPE: Web Application (Digital Wellness / Journaling / Meditation)
MISSION: Provide a serene, private space for deep reflection, self-discovery,
         and mental clarity through journaling, sound therapy, and AI guidance.

TARGET AUDIENCE:
- Individuals seeking mindfulness and self-reflection tools
- People interested in meditation and binaural beats
- Users looking for AI-guided personal growth
- Journaling enthusiasts who want a beautiful, distraction-free interface

UNIQUE VALUE PROPOSITION:
Lumina combines three powerful modalities in one cohesive experience:
1. Sacred Journaling with mood tracking
2. Scientifically-tuned binaural beats for focus/meditation
3. AI-powered oracle for gentle guidance and reflection

VALUE: Array of entry objects
  [
    {
      "id": "uuid-string",
      "text": "Entry content...",
      "mood": "calm",
      "timestamp": "2026-03-20T10:30:00.000Z"
    }
  ]

KEY: lumina_primary_intention
VALUE: String
  "My intention is to cultivate peace and clarity in my daily life."

KEY: lumina_affirmations
VALUE: Array of custom affirmations
  [
    {
      "id": 1234567890,
      "text": "I am worthy of love and success.",
      "isFavourite": true,
      "createdAt": "2026-03-15T08:00:00.000Z"
    }
  ]

KEY: lumina_daily_affirmation_index
VALUE: Integer (rotating index for daily affirmation)

KEY: lumina_affirmation_last_visit
VALUE: Date string (for streak tracking)

KEY: lumina_affirmation_streak
VALUE: Integer (consecutive days)

KEY: lumina_activity_log
VALUE: Array of activity objects
  [
    {
      "id": "uuid",
      "text": "Completed 25min focus session",
      "time": "10:30 AM"
    }
  ]

└─────────────────────────────────────────────────────────────────────────────┘

METHOD: POST
HEADERS:
  - Authorization: Bearer {VITE_GROQ_KEY}
  - Content-Type: application/json

REQUEST BODY:
{
  "model": "llama-3.1-70b-versatile",

    {
      "role": "system",

      "role": "user",

    }
  ]
}

    {
      "message": {

11. GETTING STARTED
================================================================================

PREREQUISITES:
- Node.js 18+ installed
- npm or yarn package manager
- Modern web browser (Chrome, Firefox, Edge, Safari)

INSTALLATION:

1. Clone or navigate to project directory:
   cd c:\Users\athar\OneDrive\Desktop\lumina

2. Install dependencies:
   npm install

3. Set up environment variables:
   - Copy .env.example to .env
   - Add your Groq API key:
     VITE_GROQ_KEY=your_api_key_here

4. Start development server:
   npm run dev

5. Open browser:
   - Landing page: file:///c:/Users/athar/OneDrive/Desktop/lumina/landing.html
   - Main app: http://localhost:5173

BUILD FOR PRODUCTION:

1. Create production build:
   npm run build

2. Preview production build:
   npm run preview

3. Deploy dist/ folder to hosting service
